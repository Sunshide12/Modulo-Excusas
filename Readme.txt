Para abrir correctamente los mockups se debe tener en cuenta lo siguiente.

1. Tener el xampp activado y funcionando, con el apache y el mysql activados.
2. Descomprimir el archivo .rar dentro de la carpeta htdocs que se encuentra dentro de la carpeta xampp dentro del disco local C.
3. para ver los mockups se debe escribir en el navegador de preferencia la siguiente ruta para cada uno de los mockups.


mockupDirectorUnidad = " localhost/mockupDirectorU "
mockupDocente = " localhost/mockupDocente "
mockupEstudiante = " localhost/mockupEstudiantes "
